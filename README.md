# lab_with_file

Постановка задачи: Дан файл с записанными разными символами, наша задача сжать файл следующим алгоритмом: берем 1 блок из 8 элементов, в блоке берем 1 элемент с двоичным кодом 01110101(например) и в следующие 7 элементов записываем в старший разряд соответствующий бит из 1 элемента (начиная с младшего разряда) таким образом получаем блок из 7 элементов , каждый элемент содержит соотвест бит в старшем разряде(0 или 1): 10010101.
    Чтобы расжать нужно взять блок из 7 элементов , создать буффер который будет содержать результирующий восстановленный блок из 8 элементов.в 1 элемент  достаем биты из старших разрядов , параллельно записывая элементы в буффер, а в конце записываем в финальный файл. 


тесты:
first.txt - 7 элементов: сжатый вариант - 7 элементов , расжатый вариант - 7 элементов
first.txt - 8 элементов: сжатый вариант - 7 элементов , расжатый вариант - 8 элементов
first.txt - таблица аскии , сжатый вариант - 84 элемента , расжатый вариант - 95 элементов 
если файл пустой , то сжатый тоже пустой
Если поступил текст состоящий из латиницы и хотя бы 1 кириллицы, то выводит Wrong Compress и удаляет уже заполненный файл(то есть сжатие не происходит)
Если поступает на вход большой файл, но корректно введенный, то отработает корректно
